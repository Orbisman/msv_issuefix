Sun Tranquilo Verifier is a Java tool to validate XML documents with schema language. It supports RELAX Namespace, RELAX Core, and TREX.

This product includes software developed by the Apache Software Foundation (http://www.apache.org/).

Overview

This tool is:

  1. a command line tool that can read XML documents and validate them with a schema (RELAX/TREX). If any error is found, error messages are provided.
  2. a library that can be incorporated into Java application.

For the formar usage, see commandline.html. For the latter usage, see developer.html.

Normative Information

This preview version implements

  • RELAX Core, based on the specification of JIS:TR X 0029:2000 (original Japanese text, translation to English) and additional enhancement based on the discussion held at the mailing list (relstdj, relusrj, reldeve ).
  • RELAX Namespace, based on the draft (2001-03-11 version, original Japanese text, no translation available) and additional enhancement based on the discussion held at the same mailing list.
  • TREX, based on the draft (2001-02-13 version) and the reference implementation. Only datatype vocabulary available is W3C XML Schema Part 2 Proposed Recommendation.
  • W3C XML Schema Part 2 Proposed Recommendation.

Optional Package

Apache Xerces-J Parser can be used instead of Apache Crimson Parser, which is shipped as a default parser. In that case, use of Xerces-J v.1.3.1 or later is recommended because JAXP support of eariler version is unsatisfactry.

Also, by a small modification to the source code, any JAXP1.1 compliant XML parser can be used.
