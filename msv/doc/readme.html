Sun Tranquilo Verifier is a Java tool to validate XML documents with a schema. It supports DTD, RELAX Namespace, RELAX Core, TREX, and the subset of XML Schema part 1.

This product includes software developed by the Apache Software Foundation (http://www.apache.org/).

Overview

This tool is:

  1. a command line tool that can read XML documents and validate them with a schema (DTD/RELAX/TREX/XML Schema). If any error is found, error messages are provided.
  2. a library that can be incorporated into Java application.

For the formar usage, see commandline.html. For the latter usage, see developer.html.

Normative Information

This preview version implements

  • XML DTD
  • RELAX Core, based on the specification of JIS:TR X 0029:2000 (original Japanese text, translation to English) and additional enhancements based on the discussion held at the mailing list (relstdj, relusrj, reldeve ).
  • RELAX Namespace, based on the draft (2001-03-11 version, original Japanese text, no translation available) and additional enhancements based on the discussion held at the same mailing list.
  • TREX, based on the draft (2001-02-13 version) and the reference implementation. Only datatype vocabulary available is W3C XML Schema Part 2 Proposed Recommendation.
  • Limited subset of W3C XML Schema Part 1 Proposed Recommendation. See limitations for details.
  • W3C XML Schema Part 2 Proposed Recommendation.

Optional Packages

In theory, by a small modification to the source code, any JAXP1.1 compliant XML parser can be used.

This package is shipped with Apache Xerces-J Parser v1.3.1. Eariler versions of Xerces-J and Apache Crimson Parser do not correctly implement JAXP1.1, so their use is not recommended especially when you use command-line Verifier.
