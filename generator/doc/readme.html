Sun XML Generator is a tool to generate XML instances from a RELAX or TREX grammar.

This product includes software developed by the Apache Software Foundation (http://www.apache.org/)

Quick Start

To generate a valid instance from a RELAX grammar to stdout, 

$ java -jar tranqgen.jar myGrammar.rxg

To generate 100 valid instances from a TREX pattern, 

$ java -jar tranqgen.jar -n 100 myPattern.trex out$.xml

out00.xml, out01.xml, ... out99.xml will be generated.

Tranquilo Generator is capable of generating "invalid" instances. The following example generates an instance, which is very close to valid instance. 

$ java -jar tranqgen.jar -error 1/100 mine.rxm invalid.xml

The following example generates an instance, which is almost completely wrong. 

$ java -jar traqgen.jar -error 1/2 mine.rxm terrible.xml
